## 🛠️ Getting Started
### 1.Create a folder.Open it with terminal
### 2. Clone the repository
```bash
git clone https://github.com/RTongit/Portfolio-Website.git
cd Portfolio-Website
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run the app
```bash
npm run dev
```
or, if using Create React App:
```bash
npm start
```
